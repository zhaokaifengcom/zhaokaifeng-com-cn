## 01

**题目**

设 $y=y(x)$ 是二阶常系数微分方程 $y''+py'+qy=e^{3x}$ 满足初始条件 $y(0)=y'(0)=0$ 的特解，则当 $x\to 0$，函数 $\frac{\ln(1+x^2)}{y(x)}$ 的极限 $(\quad)$

$(A)$ 不存在. $\qquad (B)$ 等于 $1$. $\qquad (C)$ 等于 $2$. $\qquad (D)$ 等于 $3$.

**知识点**

- 二阶常系数微分方程
- 极限

**答案**

(C)

**解析**

:::tip
同类题目的深度解析，[请点击这里](https://zhaokaifeng.com/23073/)
:::

因为：

$\lim_{x\to 0}\frac{\ln(1+x^2)}{y(x)}=\lim_{x\to 0}\frac{x^2}{y(x)}=\lim_{x\to 0}\frac{2x}{y'(x)}=\lim_{x\to 0}\frac{2}{y''(x)}$

将 $y(0)=y'(0)=0$ 代入方程 $y''+py'+qy=e^{3x}$ 中，得 $y''(0)=1$, 又 $y''(x)$ 在 $x=0$ 处连续, 从而

$\lim_{x\to 0}\frac{\ln(1+x^2)}{y(x)}=\lim_{x\to 0}\frac{2}{y''(x)}=\frac{2}{y''(0)}=2.$

故应选(C).

## 02

**题目**

设 $y_1, y_2$ 是一阶线性非齐次微分方程 $y'+p(x)y=q(x)$ 的两个特解，若常数 $\lambda, \mu$ 使 $\lambda y_1 + \mu y_2$ 是该方程的解，$\lambda y_1 - \mu y_2$ 是对应的齐次方程的解，则 $(\quad)$

$(A)$ $\lambda=\frac{1}{2}, \mu=\frac{1}{2}.$ $\qquad (B)$ $\lambda=-\frac{1}{2}, \mu=-\frac{1}{2}.$

$(C)$ $\lambda=\frac{2}{3}, \mu=\frac{1}{3}.$ $\qquad (D)$ $\lambda=\frac{2}{3}, \mu=\frac{2}{3}.$

**知识点**

- 一阶线性非齐次微分方程

**答案**

(A)

**解析**

由于 $y_1,y_2$ 是方程 $y'+p(x)y = q(x)$ 的解，所以：

$$
y_1'+p(x)y_1 = q(x),\ y_2'+p(x)y_2 = q(x)
$$

若 $\lambda y_1 + \mu y_2$ 是 $y'+p(x)y = q(x)$ 的解，则：

$$
\begin{aligned}
& (\lambda y_1 + \mu y_2)'+p(x)(\lambda y_1 + \mu y_2) = q(x) \\ \\
\Rightarrow \ & \lambda[y_1'+p(x)y_1] + \mu[y_2'+p(x)y_2] = q(x) \\ \\
\Rightarrow \ & (\lambda + \mu)q(x) = q(x)
\end{aligned}
$$

因 $y'+p(x)y = q(x)$ 是非齐次一阶线性微分方程，所以 $q(x) \neq 0$，从而 $\lambda + \mu = 1$. 若 $\lambda y_1 - \mu y_2$ 是 $y'+P(x)y = 0$ 解，则 $(\lambda y_1 - \mu y_2)'+p(x)(\lambda y_1 - \mu y_2) = 0$, 即：

$\lambda[y_1'+p(x)y_1] - \mu[y_2'+p(x)y_2] = 0$，也即 $(\lambda - \mu)q(x) = 0$.

从而 $\lambda - \mu = 0$，联立 $\begin{cases} \lambda + \mu = 1,\\ \lambda - \mu = 0, \end{cases}$ 解得 $\begin{cases} \lambda = \dfrac{1}{2},\\ \mu = \dfrac{1}{2}, \end{cases}$ 故应选(A).

## 03

**题目**

设非齐次线性微分方程 $y'+P(x)y=Q(x)$ 有两个不同的解 $y_1(x), y_2(x)$，$C$ 为任意常数，则该方程的通解是（　　）

(A) $C[y_1(x)-y_2(x)]$		(B) $y_1(x)+C[y_1(x)-y_2(x)]$

(C) $C[y_1(x)+y_2(x)]$		(D) $y_1(x)+C[y_1(x)+y_2(x)]$

**知识点**

- 一阶线性非齐次微分方程

**答案**

(B)

**解析**

因 $y_1(x), y_2(x)$ 是 $y'+P(x)y=Q(x)$ 的两个不同的解，故 $y_1(x)-y_2(x)$ 是齐次方程 $y'+P(x)y=0$ 的一个非零解。从而由线性方程解的结构知 $C[y_1(x)-y_2(x)]$ 是齐次方程 $y'+P(x)y=0$ 的通解。故 $y_1(x)+C[y_1(x)-y_2(x)]$ 为 $y'+P(x)y=Q(x)$ 的通解。故应选 (B)。

## 04

**题目**

若 $y=(1+x^2)^2 - \sqrt{1+x^2},\ y=(1+x^2)^2 + \sqrt{1+x^2}$ 是微分方程 $y' + p(x)y = q(x)$ 的两个解, 则 $q(x)=?$

(A) $3x(1+x^2)$	(B) $-3x(1+x^2)$	(C) $\frac{x}{1+x^2}$		   (D) $-\frac{x}{1+x^2}$

**知识点**

- 一阶线性非齐次微分方程

**答案**

(A)

**解析**

由线性方程解的结构知:

$\left[(1 + x^2)^2 + \sqrt{1 + x^2}\right] - \left[(1 + x^2)^2 - \sqrt{1 + x^2}\right] = 2\sqrt{1 + x^2}$ 为 $y' + p(x)y = 0$的解

所以 $y = \sqrt{1 + x^2}$ 也是 $y' + p(x)y = 0$ 的解

于是：$(\sqrt{1 + x^2})' + p(x)(\sqrt{1 + x^2}) = 0$，解得：

$p(x) = -\frac{1}{\sqrt{1 + x^2}} \cdot \frac{2x}{2\sqrt{1 + x^2}} = -\frac{x}{1 + x^2}$

因 $y = \sqrt{1 + x^2}$ 是 $y' + p(x)y = 0$ 的解，$y = (1 + x^2)^2 - \sqrt{1 + x^2}$ 是 $y' + p(x)y = q(x)$ 的解

所以 $(1 + x^2)^2$ 是 $y' + p(x)y = q(x)$ 的解，即：

$\left[(1 + x^2)^2\right]' - \frac{x}{1 + x^2} \cdot (1 + x^2)^2 = q(x)$

求得$q(x) = 2(1 + x^2) \cdot 2x - x(1 + x^2) = 3x(1 + x^2)$。

综上可知，本题应选 (A).

## 05

**题目**

(数一) 设曲线积分 $\int_{L} [f(x) - e^{x}] \sin y dx - f(x) \cos y dy$ 与路径无关，其中 $f(x)$ 具有一阶连续导数，且 $f(0) = 0$，则 $f(x)$ 等于（  ）

(A) $\frac{e^{-x} - e^{x}}{2}$	(B) $\frac{e^{x} - e^{-x}}{2}$	(C) $\frac{e^{x} + e^{-x}}{2} - 1$	 (D) $1 - \frac{e^{x} + e^{-x}}{2}$

**知识点**

- 一阶线性微分方程
- 曲线积分

**答案**

(B)

**解析**

由曲线积分与路径无关，得 $\frac { \partial } { \partial y } \left\{ \left[ f ( x ) - \mathrm { e } ^ { x } \right] \sin y \right\} = \frac { \partial } { \partial x } [ - f ( x ) \cos y ]$，即 $\left[ f ( x ) - \mathrm { e } ^ { x } \right] \cos y = - f ^ { \prime } ( x ) \cos y$, 化简得 $f ^ { \prime } ( x ) + f ( x ) = \mathrm { e } ^ { x }$, 等式两边同乘 $\mathrm { e } ^ { x }$, 得：$\mathrm { e } ^ { x } f ^ { \prime } ( x ) + \mathrm { e } ^ { x } f ( x ) = \mathrm { e } ^ { 2 x }$,  即 $\left[ \mathrm { e } ^ { x } f ( x ) \right] ^ { \prime } = \mathrm { e } ^ { 2 x }$, 所以 $\mathrm { e } ^ { x } f ( x ) = \frac { 1 } { 2 } \mathrm { e } ^ { 2 x } + C $. 
由 $f ( 0 ) = 0 $, 解得 $C = - \frac { 1 } { 2 } $, 因此
$f ( x ) = \frac { 1 } { 2 } \left( \mathrm { e } ^ { x } - \mathrm { e } ^ { - x } \right)$ 故应选 (B). 

注：求 $f ^ { \prime } ( x ) + f ( x ) = \mathrm { e } ^ { x }$ 也可用一阶线性微分方程公式 $f ( x ) =$ $$\mathrm { e } ^ { - \int \mathrm { d } x } \left( \int \mathrm { e } ^ { x } \cdot \mathrm { e } ^ { \int \mathrm { d } x } \mathrm { ~ d } x + C \right) .$$

## 06

**题目**

已知函数 $y = y ( x )$ 在任意点 $x$ 处的增量 $\Delta y = \frac { y \Delta x } { 1 + x ^ { 2 } } + \alpha $, 且当 $\Delta x \rightarrow 0$ 时 ， $\alpha$ 是 $\Delta x$ 的高阶无穷小， $y ( 0 ) = \pi $, 则 $y ( 1 )$ 等于

(A) $2 \pi $.	(B) $\pi$.	(C) $\mathrm { e } ^ { \frac { \pi } { 4 } }$.	(D) $\pi \mathrm { e } ^ { \frac { \pi } { 4 } } $. 

**知识点**

- 可微的定义

**答案**

(D)

**解析**

由可微的定义，知 $y ^ { \prime } = \frac { y } { 1 + x ^ { 2 } } $, 从而 $y \neq 0$ 时， $\frac { \mathrm { d } y } { y } = \frac { 1 } { 1 + x ^ { 2 } } \mathrm { ~ d } x $, 两边积分，得 $\ln | y | = \arctan x + C _ { 1 } $, 从而 $| y | = \mathrm { e } ^ { \arctan x \cdot c _ { 1 } } $, 即 $y = \pm \mathrm { e } ^ { c _ { 1 } } \cdot \mathrm { e } ^ { \arctan x } $, 令 $C = \pm \mathrm { e } ^ { c _ { 1 } } $, 则 $y = C \mathrm { e } ^ { \arctan x } $. 

又 $y ( 0 ) = \pi $, 故 $\pi = C \cdot \mathrm { e } ^ { 0 } $, 得 $C = \pi $, 从而 $y = \pi \cdot \mathrm { e } ^ { \arctan x } . $ 则 $y ( 1 ) = \pi \cdot \mathrm { e } ^ { \arctan 1 } = \pi \mathrm { e } ^ { \frac { \pi } { 4 } } $. 故应选(D).

## 07

**题目**

函数 $y = C _ { 1 } \mathrm { e } ^ { x } + C _ { 2 } \mathrm { e } ^ { - 2 x } + x \mathrm { e } ^ { x }$ 满足的一个微分方程是（  ）

(A) $y ^ { \prime \prime } - y ^ { \prime } - 2 y = 3 x \mathrm { e } ^ { x } $.	(B) $y ^ { \prime \prime } - y ^ { \prime } - 2 y = 3 \mathrm { e } ^ { x } $. 
(C) $y ^ { \prime \prime } + y ^ { \prime } - 2 y = 3 x \mathrm { e } ^ { x } $.	(D) $y ^ { \prime \prime } + y ^ { \prime } - 2 y = 3 \mathrm { e } ^ { x } $. 

**知识点**

- 二阶常系数微分方程

**答案**

(D)

**解析**

由微分方程的通解 $y = C _ { 1 } \mathrm { e } ^ { x } + C _ { 2 } \mathrm { e } ^ { - 2 x } + x \mathrm { e } ^ { x }$ 知， $\mathrm { e } ^ { x } , \mathrm { e } ^ { - 2 x }$ 是齐次方程的2 个无关的解， $x \mathrm { e } ^ { x }$ 为非齐次方程的解.故 $r _ { 1 } = 1 , r _ { 2 } = - 2$ 是特征方程的两个根，故特征方程为 $( r - 1 ) ( r + 2 ) = 0 $, 即 $r ^ { 2 } + r - 2 = 0 $, 从而齐次方程为 $y ^ { \prime \prime } + y ^ { \prime } - 2 y = 0 $. 设非齐次方程为 $y ^ { \prime \prime } + y ^ { \prime } - 2 y = f ( x ) $, 因 $x \mathrm { e } ^ { x }$ 为非齐次方程的解，于是：

 $\left( x \mathrm { e } ^ { x } \right) ^ { \prime \prime } + \left( x \mathrm { e } ^ { x } \right) ^ { \prime } - 2 x \mathrm { e } ^ { x } = f ( x ) ,$ 即 $( x + 2 ) \mathrm { e } ^ { x } + ( x + 1 ) \mathrm { e } ^ { x } - 2 x \mathrm { e } ^ { x } = f ( x ) $, 
从而 $f ( x ) = 3 \mathrm { e } ^ { x } $, 则所求微分方程为 $y ^ { \prime \prime } + y ^ { \prime } - 2 y = 3 \mathrm { e } ^ { x } $. 

综上可知，本题应选 (D).

## 08

**题目**

微分方程 $y ^ { \prime \prime } + y = x ^ { 2 } + 1 + \sin x$ 的特解形式可设为
(A) $y ^ { * } = a x ^ { 2 } + b x + c + x ( A \sin x + B \cos x ) $. 
(B) $y ^ { * } = x \left( a x ^ { 2 } + b x + c + A \sin x + B \cos x \right) .$ 
(C) $y ^ { * } = a x ^ { 2 } + b x + c + A \sin x $. 
(D) $y ^ { * } = a x ^ { 2 } + b x + c + A \cos x $.

- 二阶常系数微分方程

## 09

微分方程 $y ^ { \prime \prime } - \lambda ^ { 2 } y = \mathrm { e } ^ { \lambda x } + \mathrm { e } ^ { - \lambda x } ( \lambda > 0 )$ 的特解形式为
(A) $a \left( \mathrm { e } ^ { \lambda x } + \mathrm { e } ^ { - \lambda x } \right) .$		(B) $a x \left( \mathrm { e } ^ { \lambda x } + \mathrm { e } ^ { - \lambda x } \right) .$

- 二阶常系数微分方程

**答案**

(C)

**解析**

$y ^ { \prime \prime } - \lambda ^ { 2 } y = 0$ 的特征：方程为 $r ^ { 2 } - \lambda ^ { 2 } = 0 $, 解得特征根为 $r _ { 1 . 2 } = \pm \lambda $. 故 $y ^ { \prime \prime } - \lambda ^ { 2 } y = \mathrm { e } ^ { \lambda x }$ 的特解可设为 $y _ { i } ^ { * } = a x \mathrm { e } ^ { \lambda x } $, $y ^ { \prime \prime } - \lambda ^ { 2 } y = \mathrm { e } ^ { - \lambda x }$ 的特解可设为 $y _ { 2 } ^ { * } = b x \mathrm { e } ^ { - \lambda x }$,

$$y _ { i } ^ { * } + y _ { 2 } ^ { * } = x \left( a \mathrm { e } ^ { \lambda x } + b \mathrm { e } ^ { - \lambda x } \right) .$$ 

所以应选(C).

## 10

$$y ^ { * } = y _ { i } ^ { * } + y _ { 2 } ^ { * } = A \mathrm { e } ^ { 2 x } + x \mathrm { e } ^ { 2 x } ( B \cos 2 x + C \sin 2 x ) ,$$ 

综上可知，本题应选 (C).

## 13

**题目**

设 $y = \frac { 1 } { 2 } \mathrm { e } ^ { 2 x } + \left( x - \frac { 1 } { 3 } \right) \mathrm { e } ^ { x }$ 是二阶常系数非齐次线性微分方程 $y ^ { \prime \prime } + a y ^ { \prime } + b y = c \mathrm { e } ^ { x }$ 的一个特解，则

(A) $a = - 3 , b = 2 , c = - 1 $.	(B) $a = 3 , b = 2 , c = - 1 $. 

(C) $a = - 3 , b = 2 , c = 1 $.	    (D) $a = 3 , b = 2 , c = 1 $. 

**知识点**

- 二阶常系数非齐次微分方程

**答案**

(A)

**解析**

因方程右端白由项为 $c \mathrm { e } ^ { x } $, 故 $\frac { 1 } { 2 } \mathrm { e } ^ { 2 x }$ 为对应齐次方程的解，从而 $r _ { 1 } = 2$ 为特征方程的一个根；于是 $x \mathrm { e } ^ { x }$ 一定不是齐次方程的解，否则 $r = 1$ 应是特征方程的二重根，故 $x \mathrm { e } ^ { x }$ 为非齐次方程的解；再由二阶常系数非齐次线性微分方程解的结构知 $- \frac { 1 } { 3 } \mathrm { e } ^ { x }$ 应是对应齐次方程的解，从而 $r _ { 2 } = 1$ 为特征方程的另一个根.故特征方程为 $( r - 1 ) ( r - 2 ) = 0 $, 即 $r ^ { 2 } - 3 r + 2 = 0 $, 则对应的齐次方程为 $y ^ { \prime \prime } - 3 y ^ { \prime } + 2 y = 0 $. 即 $a = - 3 , b = 2 $. 

将 $x \mathrm { e } ^ { x }$ 代入 $y ^ { \prime \prime } - 3 y ^ { \prime } + 2 y = c \mathrm { e } ^ {x}$ 中，有 $\left( x \mathrm { e } ^ { x } \right) ^ { \prime \prime } - 3 \left( x \mathrm { e } ^ { x } \right) ^ { \prime } + 2 x \mathrm { e } ^ { x } = c \mathrm { e } ^ { x } ,$ 于是有 $x + 2 - 3 ( x + 1 ) + 2 x = c $, 从而 $c = - 1 $. 

综上可知，本题应选 (A).

## 14

设非齐次方程的特解为 $y ^ { * } = A \mathrm { e } ^ { 2 x } $, 代入 $y ^ { \prime \prime } - 4 y ^ { \prime } + 3 y = 2 \mathrm { e } ^ { 2 x }$ 中，有

$$4 A \mathrm { e } ^ { 2 x } - 8 A \mathrm { e } ^ { 2 x } + 3 A \mathrm { e } ^ { 2 x } = 2 \mathrm { e } ^ { 2 x } ,$$ 
故 $- A = 2 $, 得 $A = - 2 $, 则 $y ^ { * } = - 2 \mathrm { e } ^ { 2 x } $. 从而方程 $y ^ { \prime \prime } - 4 y ^ { \prime } + 3 y = 2 \mathrm { e } ^ { 2 x }$ 的通解为
 $y = \tilde { y } + y ^ { * } = C _ { 1 } \mathrm { e } ^ { x } + C _ { 2 } \mathrm { e } ^ { 3 x } - 2 \mathrm { e } ^ { 2 x } $, 其中 $C _ { 1 } , C _ { 2 }$ 为任意常数.

## 45

**题目**

若函数 $f ( x )$ 满足方程 $f ^ { \prime \prime } ( x ) + f ^ { \prime } ( x ) - 2 f ( x ) = 0$ 及 $f ^ { \prime \prime } ( x ) + f ( x ) = 2 \mathrm { e } ^ { x } $, $f ( x ) = ?$ 

**知识点**

- 二阶常系数齐次微分方程

**答案**

$\mathrm { e } ^ { x }$

**解析**
二阶常系数齐次方程 $f ^ { \prime \prime } ( x ) + f ^ { \prime } ( x ) - 2 f ( x ) = 0$ 的特征方程为 $r^{2} + r - 2 = 0$, 解之得特征根为： $r _ { 1 } = 1 , r _ { 2 } = - 2 $.

从而 $f ^ { \prime \prime } ( x ) + f ^ { \prime } ( x ) - 2 f ( x ) = 0$ 的通解为： $f ( x ) = C _ { 1 } \mathrm { e } ^ { x } + C _ { 2 } \mathrm { e } ^ { - 2 x } $. 

又因为 $f ( x )$ 满足 $f ^ { \prime \prime } ( x ) + f ( x ) = 2 \mathrm { e } ^ { x } $, 故：
$$\left( C _ { 1 } \mathrm { e } ^ { x } + C _ { 2 } \mathrm { e } ^ { - 2 x } \right) ^ { \prime \prime } + C _ { 1 } \mathrm { e } ^ { x } + C _ { 2 } \mathrm { e } ^ { - 2 x } = 2 \mathrm { e } ^ { x } ,$$ 
整理有 $2 C _ { 1 } \mathrm { e } ^ { x } + 5 C _ { 2 } \mathrm { e } ^ { - 2 x } = 2 \mathrm { e } ^ { x } $, 解得 $C _ { 1 } = 1 , C _ { 2 } = 0 $,  

综上可知，$f ( x ) = \mathrm { e } ^ { x } $.
